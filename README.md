# 🛡️ AI Safety Guardian

An AI-powered real-time safety system that monitors chat, video, and audio interactions to protect children from unsafe content and behavior.

## 🚀 Key Features
- Context-aware chat analysis
- NSFW video detection
- Speech-to-text monitoring
- Unified risk scoring system
- Real-time alerts & intervention

## 🧠 Tech Stack
Python, OpenCV, NLP, Transformers, Whisper, Flask
